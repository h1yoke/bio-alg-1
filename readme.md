# Алгоритмы в биоинформатике 1

* parser.cpp, parser.h - Общий код с парсером FASTQ
* fastq-stats/main.cpp - Решение для шага 1
* gc-stats/main.cpp - Решение для шага 2
* phred-stats/main.cpp - Решение для шага 3
* trimming-stats/main.cpp - Решение для шага 4

В каждой задаче входной FASTQ файл читается и парсится построчно
с помощью `parse_fastq` в структуру FastqRecord

Для удобства в корне лежит `run.sh` скрипт, который по очереди 
компилирует и запускает все шаги через g++
